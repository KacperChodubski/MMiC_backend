# README

## Pobranie

```
git clone https://github.com/KacperChodubski/MMiC_backend.git
cd MMiC_backend
chown u+x ./utils.py
```

## Obsługa

### Uruchomienie

```
./utils.py up
```

### Zamknięcie

```
./utils.py down
```

### Logi

```
./utils.py logs
```

## Dokumentacja

Dokumentację naszego API znajdziemy na http://localhost:8080/docs
